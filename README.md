# Devina Archives - Music Production Company Website

A professional, responsive static HTML website for Devina Archives, a premium music production and audio engineering company.

## 🎵 Features

- **Responsive Design**: Fully mobile-friendly website that works on all devices
- **Professional Aesthetic**: Clean design using offwhite, cream, and black color scheme
- **Multiple Pages**:
  - Home page with hero section and features
  - Portfolio showcase with sample projects
  - Services page with detailed offerings and pricing
  - About page with team and company information
  - Contact page with form and frequently asked questions

- **Interactive Elements**:
  - Smooth navigation and scrolling
  - Form validation and submission
  - Animated counters for statistics
  - Fade-in animations on scroll
  - Hover effects and transitions

## 🛠️ Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with CSS Grid and Flexbox
- **JavaScript** - Vanilla JS for interactivity

## 🚀 Getting Started

### Prerequisites
- A web browser
- Git (for cloning)
- GitHub account (already set up)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/davidthapa0431/devina-archives.git
cd devina-archives
```

2. Open in your browser:
   - Simply open `index.html` in your browser
   - Or serve using a local web server

### Using GitHub Pages

This site is set up for GitHub Pages deployment:

1. Go to your repository settings
2. Navigate to "Pages" section
3. Set source to `main` branch
4. Save and wait for deployment
5. Your site will be available at: `https://davidthapa0431.github.io/devina-archives/`

## 🔧 Customization

### Update Contact Information
Edit `contact.html`:
- Email: Replace `info@devinaarchives.com`
- Phone: Replace `+1 (234) 567-890`
- Address: Update location details
- Hours: Modify business hours

### Add Social Media Links
Update all pages' footer sections:
- Instagram, Twitter, YouTube, Spotify links

### Modify Portfolio
In `portfolio.html`:
- Replace placeholder images with actual artwork
- Update artist names and project descriptions
- Add new portfolio items by duplicating the `.portfolio-item` div

### Update Pricing
In `services.html`:
- Modify pricing tiers and amounts
- Update included features
- Add or remove pricing plans

### Change Colors
In `style.css`:
- Update CSS variables in `:root` selector
- All colors are centralized for easy customization

## 🐛 Troubleshooting

### Form Not Working
- Check browser console for errors
- Ensure JavaScript is enabled
- Currently form displays success message (no backend integration)

### Navigation Issues
- Clear browser cache
- Check that all HTML files are in the same directory
- Ensure file paths in links are correct

### Styling Issues
- Verify `style.css` is linked in all HTML files
- Check browser compatibility
- Try different browser if CSS doesn't load

## 🔒 Security Notes

- Contact form is client-side only (no server backend)
- For production, implement backend form handling
- Add HTTPS when deploying
- Sanitize any user inputs if adding server-side processing
